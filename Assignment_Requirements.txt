# Part 1 – Pose SLAM Theory

Assume:

* Initial pose prior:

[
p(x_0)=\mathcal{N}(\hat{x}_0,\Sigma_0)
]

* Noisy odometry measurements:

[
z_k = x_{k+1} \ominus x_k + v_k
]

where

[
v_k \sim \mathcal{N}(0,\Sigma_v)
]

## Question 1

Express the joint posterior as a product of:

* Odometry terms:

[
p(z_k \mid x_k, x_{k+1})
]

* Prior term:

[
p(x_0)
]

No need to specify the exact PDF.

The probabilistic terms composing the posterior should be identified as **factors**.

## 2.2 Construct the Factor Graph

The variable:

```text
dposes
```

contains noisy relative pose measurements between consecutive states.

These correspond to:

[
z_k
]

with covariance:

[
\Sigma_v
]

---

### (a) Create the Graph

```matlab
graph = gtsam.NonlinearFactorGraph;
```

Add factors using:

```matlab
graph.add(factor);
```

---

### (b) Add Between Factors

Use:

```matlab
gtsam.BetweenFactorPose3(
    key1,
    key2,
    relative_pose,
    noise_model)
```

Create the noise model:

```matlab
gtsam.noiseModel.Diagonal.Sigmas([Sr, St])
```

Use:

```text
Rotation std:     1e-3 rad
Translation std:  0.1 m
```

for each axis.

The covariance matrix is:

[
\Sigma_v=
\begin{bmatrix}
\operatorname{diag}(S_r^2) & 0\
0 & \operatorname{diag}(S_t^2)
\end{bmatrix}
]

where the square is applied element-wise.

---

### (c) Add Prior Factor

Assume:

* Initial position is at the origin.
* Robot axes are aligned with the global frame.

Use:

```matlab
gtsam.PriorFactorPose3(
    key,
    pose,
    noise_model)
```

Use the same covariance as before:

[
\Sigma_0=\Sigma_v
]
